## Task 4：群体特征构建与演化分析

### Task 4.1 群体特征聚合
- [ ] 实现均值池化聚合
- [ ] 实现注意力池化聚合（可学习权重）
- [ ] 实现图池化方法（DiffPool/SAGPool）
- [ ] 对比不同聚合方式的群体表示质量

### Task 4.2 群体交互图构建
- [ ] 统计群体间的交互数量（转发+回复）
- [ ] 构建群体间交互强度矩阵
- [ ] 计算群体间交互的语义特征
- [ ] 生成群体交互图（PyG格式）

### Task 4.3 演化分析

#### 4.3.1 时序立场追踪(Temporal Stance Tracking)
- [ ] 将55个时间切片划分为时间窗口（每5个切片为一窗口，共11窗口）
- [ ] 利用TGAN时态嵌入，通过分类器得到每个个体的softmax立场概率（连续立场强度）
- [ ] 按群体聚合，计算每个群体在每个时间窗口的平均立场强度
- [ ] 追踪并绘制群体立场强度随时间的变化轨迹

#### 4.3.2 交互-演化关联分析(Interaction-Evolution Correlation)
- [ ] 统计时间窗口t内群体A→B的交互量（转发数+回复数）
- [ ] 计算时间窗口t+1内群体B的立场变化量（Δstance）
- [ ] 计算交互量与后续立场变化的Pearson/Spearman相关性
- [ ] 区分同立场交互 vs 跨立场交互的不同影响效果
- [ ] 假设验证：跨立场交互→立场趋同？同立场交互→回音室强化？

#### 4.3.3 群体演化模式分类
- [ ] 定义四种演化模式：趋同型/极化型/稳定型/传染型
- [ ] 对每对群体交互进行演化模式分类
- [ ] 统计各演化模式的分布情况

#### 4.3.4 影响力传播建模
- [ ] 实现DeGroot模型：x_i(t+1) = Σ_j w_ij * x_j(t)
- [ ] 实现Friedkin-Johnsen模型：加入固执系数
- [ ] 实现基于GNN的演化预测（输入t时刻特征+图，预测t+1立场）
- [ ] 对比三种模型的预测效果（MAE/RMSE）

#### 4.3.5 Granger因果分析
- [ ] 对每对群体(A,B)检验A→B交互是否Granger-causes B的立场变化
- [ ] 构建因果影响网络（有向图）
- [ ] 识别核心影响群体（出度最高的群体）

#### 4.3.6 动态网络演化分析
- [ ] 将群体交互图按时间窗口切片，得到时序图序列
- [ ] 分析每个时间点的网络拓扑变化（密度、路径长度、社区结构）
- [ ] 检测交互模式突变点
- [ ] 分析突变前后群体立场的变化

### Task 4.4 可视化与评价指标
- [ ] 绘制演化轨迹图(Evolution Trajectory)
- [ ] 绘制烟花图(Firework Diagram)：同立场群体内部信息扩散
- [ ] 绘制演化热力图(Evolution Heatmap)

- [ ] 计算群体极化指数（E-I Index时序）
- [ ] 计算极小化指标（群体合并合理性评估）
- [ ] 计算极大化指标（群体分裂合理性评估）
- [ ] 计算演化预测准确率（MAE/RMSE）
- [ ] 计算因果显著性（p<0.05的因果对占比）
- [ ] 计算立场稳定性指数（时序方差）
- [ ] 总结群体间交互如何影响立场演化的关键发现
